# GCoffee-Shop-Analysis

Analize the Sales Performance of coffee house named Gcoffee in Q2 of 2025 period using several analysis method such as cohort and market basket analysis. Source of dataset is from kaggle:
https://www.kaggle.com/datasets/geraldooizx/g-coffee-shop-transaction-202307-to-202506. 

Analize processed by SQL and pyhton. You can take a look a visualization result in tableau public:
https://public.tableau.com/app/profile/fahrul.rahman/viz/SalesPerformanceofGcoffeeinQ22025/Dashboard1
